# Horn Lake Fire/EMS App

This is a standalone React/Vite app converted from the original Base44 export.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Render Settings

Use these settings for a Render Web Service:

```text
Build Command: npm install && npm run build
Start Command: npm start
```

The app no longer depends on the Base44 SDK or Base44 Vite plugin. Some images
and documents may still be linked from public `media.base44.com` URLs until
those assets are downloaded and rehosted.
